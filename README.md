## ğŸ›  Usage Instructions

### iOS
Place the icon set inside your Xcode project's `Assets.xcassets/AppIcon.appiconset/`.

### Android
- Add icons to: `android/app/src/main/res/mipmap-*/`
- Adaptive icons go to: `android/app/src/main/res/mipmap-anydpi-v26/`

### Web
Include this in the HTML `<head>`:

```html
<link rel="icon" href="/icons/favicon.ico" />
<link rel="apple-touch-icon" href="/icons/apple-icon.png" />
```
